# Mohid Watch React

Mohid Watch is a modern web application designed to provide a seamless shopping experience for watch enthusiasts. Built with a robust tech stack, it offers a responsive and user-friendly interface.

## Features

- **Working Cart**: Add, remove, and manage items in your cart effortlessly.
- **Working Carousel**: A dynamic carousel showcasing featured products.
- **Responsive Navigation Menu**: Fully functional navigation menu optimized for mobile devices.
- **Subscription Form**: Subscribe to updates via the contact section on the home page.

## Pages

- **Home**: The main landing page with featured products and a subscription form.
- **Cart**: View and manage items added to your cart.
- **404**: A custom page for handling non-existent routes.

## Tech Stack

- **Frontend Framework**: [React](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Type Checking**: [TypeScript](https://www.typescriptlang.org/)
- **Routing**: [React Router DOM](https://reactrouter.com/)
- **HTTP Client**: [Axios](https://axios-http.com/)
- **API**: [DummyJSON API](https://dummyjson.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Notifications**: [React Hot Toast](https://react-hot-toast.com/)
- **Unique Identifiers**: [UUID](https://www.npmjs.com/package/uuid)
- **Linting**: [ESLint](https://eslint.org/)
